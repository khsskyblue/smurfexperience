스머프 마을체험전 고양꽃전시관 

대중교통 이용시

지하철

- 정발산역(3호선 대화행)하차, 1번 출구 호수공원 방향 도보 10분거리

버스

- 일산동구청(정발산역) 하차, 호수공원 방향 도보 10분거리

3300(인천공항) 33, 150(김포공항) 770, 7727(신촌) 830, 9707(영등포)

200(합정) 706, 1000, 2000, 8880, 9701, 9714(서울역), M7106, 108(여의도)

9700, 9711(강남역)

- 호수공원 하차 도보 3분거리 3000(인천)

고속버스

- 고양시 고양종합터미널(백석역) 또는 화정터미널(화정역) → 지하철 3호선 정발산역 1번 출구 → 호수공원 방향 도보 10분

기차

- 서울역 → 지하철 1호선 승차 → 종로3가 3호선(대화행)환승 → 정발산역 하차 → 1번출구 호수공원 방향 도보 10분

- 경의선 → 대곡역 하차 후 지하철 3호선 정발산역 하차, 1번 출구 호수공원 방향 도보 10분

자가용 이용시

수도권

- 강변북로 → 자유로 → 장항IC → 고양꽃전시관

- 올림픽대로 → 행주대교 → 자유로 → 장항IC → 고양꽃전시관

- 외곽순환고속도로 → 김포대교 → 자유로 → 장항IC → 고양꽃전시관

지방

- 대전 : 신탄진 IC → 경부고속도로 → 88올림픽대로 → 행주대교 → 자유로 → 장항IC → 고양꽃전시관

- 대구 : 구마고속도로 → 경부고속도로 → 88올림픽대로 → 행주대교 → 자유로 → 장항IC → 고양꽃전시관

- 광주 : 호남고속도로 → 천안논산고속도로 → 경부고속도로 → 88올림픽대로 → 행주대교 → 자유로 → 장항IC →고양꽃전시관

- 부산 : 남해고속도로 → 대구부산고속도로 → 경부고속도로 → 88올림픽대로 → 행주대교 → 자유로 → 장항IC →고양꽃전시관

고속도로

- 중부, 경부고속도로 : 88올림픽대로 → 행주대교 → 자유로 → 장항IC → 고양꽃전시관

- 서해안 고속도로 : 조남I.C →외곽순환도로 → 김포대교 → 자유로 → 장항IC → 고양꽃전시관

주차장 안내

호수공원 제 1, 2, 3, 4 주차장

- 호수공원 제 1, 2, 3, 4 주차장은 꽃박람회와 무관하며, 별도의 주차 요금이 징수됨을 알려드립니다.
